# ❤️ Heart Disease Prediction using Machine Learning

![Python](https://img.shields.io/badge/Python-3.11-blue)

A Machine Learning-powered web application that predicts the likelihood of heart disease based on a patient's clinical parameters. The application is built with **Streamlit** and uses a **Logistic Regression model optimized with GridSearchCV** to provide real-time predictions.

## 🚀 Live Demo

🔗 **Streamlit App:**  
https://heartdiseasepredictionmodel-hqbeblvlyjwqb7ekrwnjjw.streamlit.app/

---

## 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early prediction can help individuals seek timely medical attention.

This application allows users to enter important medical information and instantly predicts whether they are at **High Risk** or **Low Risk** of heart disease using a trained Machine Learning model.

---

## ✨ Features

- Interactive and user-friendly Streamlit interface
- Real-time heart disease prediction
- Logistic Regression model with hyperparameter tuning using GridSearchCV
- Displays prediction probability using `predict_proba()`
- Uses 13 important clinical parameters
- Clean and responsive web interface

---

## 🧠 Machine Learning Model

**Algorithm Used**

- Logistic Regression
- GridSearchCV for Hyperparameter Tuning

**Model Performance**

- Accuracy: **88.5%**
- Cross Validation: 5-Fold Cross Validation

---

## 📊 Input Features

The prediction is based on the following medical parameters:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG Results
- Maximum Heart Rate Achieved
- Exercise-Induced Angina
- ST Depression (Oldpeak)
- Slope of Peak Exercise ST Segment
- Number of Major Vessels Colored (CA)
- Thalassemia

---

## 📈 Prediction Output

The application provides:

- Heart Disease Risk Prediction
  - ✅ Low Risk
  - ⚠️ High Risk

- Prediction Probability
  - Probability of No Disease
  - Probability of Disease

## 📥 Installation

Clone the repository

```bash
git clone https://github.com/samikshahujare/heart_disease_prediction_model.git
```

Move into the project directory

```bash
cd heart_disease_prediction_model
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run heart.py
```
